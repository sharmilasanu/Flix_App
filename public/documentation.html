My Flix API

My flix API has the following essential fetaures

Essential Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

Resource URL

Visit W3Schools.com!
Request URL Responses
GET /movies Return a list of ALL movies.
GET /movies/:title Return data about a single movie by title.
GET /movies/:genre Return data by genre.
GET /director/:name Return data about a director by name.
POST /users Allow new users to register.
PUT /users/:id/:user_info Allow user to update the user info
POST /users/:id/:favorites Allow user to add a movie to the list of favorites.
DELETE /users/:id/:favorites All user to remove a movie from the list of favorites.
DELETE /users/:id Allow existing user to deregister.
